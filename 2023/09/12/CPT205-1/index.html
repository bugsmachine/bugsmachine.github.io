CPT205 Computer Graphics 1: General Introduction to Hardware and Software

Computer Graphics is concerned with all aspects of producing pictures or images using a computer including the image you see on the computer screen,the programming code used to create images and mathematical models of real-world (Virtual World).

Hardware and Software

  • Graphics Hardware

    • Input (Light Pens…) -> Processing (GPU) -> Output Devices (LCD…)
    • Framebuffers: Block if memory, hold all the display content
    • Pixels (an element of framebuffer) and Screen Resolution

  • Graphics Software

    • Techniques (Algorithms, Procedures)
    • Programming Library / API (OpenGL, JOGL and so on)

Framebuffer

If the screen resolution is 640 × 480, it means the screen has 640 pixels in horizontal and 480 in vertical. Thus, the framebuffer’s capacity will be 640 × 480 × bit depth (number of bits allocated per pixel in a buffer).

If bit depth is 1, that means each pixel only has 1 bit and indicate the screen only can show black or white because 1 bit can only have 0 or 1. Usually, 32 bits (RGBA) are needed to give a colorful screen.

RGBA (32 bits) = Red (8 bits) Green (8 bits) Blue (8 bits) Alpha (8 bits)

Total 16,777,216 colours, potential for 255 reds, greens and blues. Alpha indicates the transparency degree which is in range 0-1, 0 stands for totally transparency and 1 for fully opaque.

To show the graphics, the GPU need have 640 × 480 × 32 bits = 1,228,800 bytes framebuffer storage for a 640 × 480’s screen resolution.

Pixels

Pixel is the most basic addressable image element in a screen. It can be CRT (Colour triad -RGB phosphor dots) or LCD (Single colour element).

Raster display

  1. Video Formats:

    • Interlaced: NTSC, PAL….
    • Non-interlaced: VAG, SVAG…
    • RGB: 3 independent video signals and synchronization signal, vary in resolution and refresh rate

    In an interlaced display or image, the content is split into two or more sets of lines, and these sets are displayed or processed alternately, creating the appearance of a complete image or video frame.

    This process happens rapidly, creating the illusion of a complete image to the human eye.

  2. Display Equipment:

    • Cathode ray tubes (CRTs):
      • Strong electrical fields and high voltage
      • Very good resolution
      • Heavy, not flat
    • Liquid crystal displays (LCDs):
      • Flat
      • Light weight
      • Low power consumption

    LCDs elements do not emit light; use backlight behind the LCD matrix. Colour is obtained by placing filters in front of each LCD element. Voltage degree will change the grey degree of LCDs elements.

Graphics Software

Graphics software include graphics drivers, libraries, editors, geometric modellers….

Some programming API can help to program such as OpenGL, JOGL…
